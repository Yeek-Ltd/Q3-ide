## <a id="reh"></a>What are reh and reh-web archives?

- Remote Host (`reh`) is the server component for remote ssh/wsl which runs it on a "remote" computer and makes that remote computer accessible via Q3 IDE.
- Web Host (`reh-web`) is the server component of the command `q3ide serve-web` which runs it locally and makes Q3 IDE accessible via a browser.
